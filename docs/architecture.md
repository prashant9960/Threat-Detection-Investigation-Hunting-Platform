# Threat Detection, Investigation & Hunting Platform

## Objective

This project simulates a real Security Operations Center (SOC) environment.

The platform is designed to detect, investigate, hunt, and respond to simulated cyber attacks using enterprise security monitoring techniques.

## Components

### Windows 11 Workstation (WS-01)

Represents an employee workstation.

### Kali Linux (ATTACKER)

Represents an external attacker.

### Splunk Enterprise (SPLUNK)

Acts as the Security Information and Event Management (SIEM) platform.

### Sysmon

Provides endpoint telemetry and detailed event logging.

## Project Goals

* Detection Engineering
* Incident Investigation
* Threat Hunting
* Security Automation
* MITRE ATT&CK Mapping

## Planned Attack Campaigns

1. Password Spray
2. PowerShell Execution
3. Persistence Techniques
4. Discovery Activities
5. Data Collection
